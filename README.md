# java_multithread

 
• Processus :

- application ,  programme  
- environnement d’exécution auto-contenu (mémoire, etc.)

• Thread(s) :

- processus léger  
- rattachée(s) à un processus

- différents branches d’exécution
- gerer par le JVM
